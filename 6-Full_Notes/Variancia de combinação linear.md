# Variancia de combinação linear

Os slides apresentam a fórmula:

$$
Var(aX + bY) = a^2Var(X) + b^2Var(Y) + 2abCov(X, Y)
$$
e, se $X$ e $Y$ forem independentes, a covariância é zero, então fica


$$
Var(aX + bY) = a^2Var(X) + b^2Var(Y)
$$

## Intuição

Quando você soma duas variáveis, o espalhamento total depende de:

- quanto cada uma se espalha sozinha;
- e de como elas variam juntas.

Essa segunda parte é exatamente a covariância.

Se elas “andam juntas”, isso altera a variância da soma.  
Se são independentes, esse efeito desaparece.
